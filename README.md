# 🌐 Personal Developer Portfolio

A modern, responsive, and interactive **frontend developer portfolio** built using React.js and Tailwind CSS.  
It showcases my skills, experience, and projects in a visually engaging UI with smooth interactions and animations.

## 🚀 Live Demo  

https://sankari-d27.github.io/React-Portfolio/

## 🛠️ Tech Stack

- ⚛️ React.js (Functional Components, Hooks)
- 🎨 Tailwind CSS (Modern UI styling)
- 🧠 JavaScript (ES6+)
- 🧱 HTML5 & CSS3
- 🔀 React Router (section navigation via anchors)
- 🎯 React Icons
- 📦 Vite / Create React App (depending on your setup)

## ✨ Features

- 📱 Fully responsive design (mobile + desktop)
- 🎯 Smooth scroll navigation (Home, About, Projects, Contact)
- 💼 Dynamic Experience timeline UI
- 🧩 Interactive “toggle” experience slider
- 🎨 Modern gradient UI with glassmorphism effects
- 🧑‍💻 Projects showcase with live & GitHub links
- 📄 Resume download option
- ⚡ Fast and lightweight React SPA
